\# Estudos em Python



Repositório com exercícios desenvolvidos durante o aprendizado dos fundamentos da linguagem Python, com foco em lógica de programação e conceitos básicos da linguagem.



Os códigos abordam conceitos essenciais para iniciantes, como:



\- Definição e utilização de funções

\- Estruturas condicionais (`if`, `elif`, `else`)

\- Estruturas de repetição (`for`, `while`, `break`)

\- Operadores matemáticos e lógicos

\- Entrada e saída de dados no terminal

\- Conversão de tipos (`int`, `float`, `str`)

\- Uso de listas e dicionários

\- Parâmetros e retorno de funções

\- Organização e indentação do código

\- Interação com o usuário via linha de comando



Os exercícios incluem, entre outros:



\- Calculadora básica

\- Verificação se um número é par ou ímpar

\- Comparação de valores

\- Sistemas simples com entrada do usuário

\- Manipulação de listas e dicionários

\- Exemplos funcionais para prática de lógica



Os arquivos estão organizados em pastas temáticas para facilitar o estudo e a navegação.



Projeto criado para fins de estudo e prática dos fundamentos de Python.
